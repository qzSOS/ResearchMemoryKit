# Decision Log

> Append-only. Add a new entry for each important decision. Do not silently rewrite old rationale.

## D-001: Project Scope

| Field | Value |
|---|---|
| Date | YYYY-MM-DD |
| Status | Active |
| Question | What is the project trying to solve? |
| Decision | Short answer. |
| Rationale | Why this choice is better than alternatives. |
| Alternatives | Options that were considered. |
| Revisit condition | Evidence that would cause this decision to change. |
| Source | Link to related file, issue, experiment, or commit. |
