# Toy Research Project Agent Instructions

## Startup

1. Read `memory/CURRENT_STATE.md`.
2. Read `memory/README.md`.
3. Inspect git status.
4. Before running an experiment, register it under `registry/`.

## Rules

- Keep current facts in `memory/CURRENT_STATE.md`.
- Keep this router free of mutable results.
- Log failures in `memory/FAILED_ATTEMPTS.md`.
- Add reproducible bugs to `memory/PITFALLS.md`.
- An experiment is not complete until the Memory Completion Gate in `memory/WORKFLOW.md` is satisfied.
